Elementary Logic: Chapter 4 Solutions

In all of the exercises that follow, when asked to give an interpretation, I have taken the liberty to assign denotations only to the non-logical constants that appear in the sentences under consideration. As you might remember, Mates defines an interpretation such that all of the non-logical constants of L are assigned denotations.

Exercise 1

Let ψ be any sentence, and let -φ be a valid sentence. -φ is true under every interpretation since it is valid. It follows that φ must be false under every interpretation since it is the negation of -φ. Clearly, there can be no interpretation under which φ is true and ψ is false, since φ can not be true under any interpretation. Therefore, ψ is a consequence of φ and since ψ can be any sentence at all, every sentence is a consequence of φ.

Exercise 2

Let θ be any sentence, and suppose {φ,ψ} is a set of inconsistant sentences. There is no interpretation for which both φ and ψ are true since they are inconsistant. Clearly, there can be no interpretation for which both φ and ψ are true and θ is false since φ and ψ are both true under no interpretation. Therefore, θ is a consequence of {φ,ψ}, and since θ can be any sentence, every sentence is a consequence of {φ,ψ}.

Exercise 3

Let Γ be any set of sentences and φ be a valid sentence. φ is true under every interpretations since it is valid. Trivially, φ is false under no interpretations. Clearly, there is no interpretation under which all the sentences of Γ are true and φ is false since φ is false under no interpretations. Therefore, φ is a consequence of Γ and since Γ can be any set of sentences, φ is a consequence of every set of sentences.

Exercise 4

Let φ and ψ be sentences. ψ is a consequence of φ if and only if there is no interpretation under which φ is true and ψ is false. This holds if and only if there is no interpretation under which (φ → ψ) is false. Said another way, this holds if and only if (φ → ψ) is true under every interpretation. Therefore, ψ is a consequence of φ if and only if (φ → ψ) is valid.

Exercise 5

If -φ is a valid sentence then -φ is true under every interpretation and φ is false under every interpretation. Said another way, there is no interpretation under which φ is true. Therefore, there is no interpretation under which all the sentences of {φ} are true so, φ is inconsistant with itself (and with any other set of sentences it is included in).
